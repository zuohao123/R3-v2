# 论文配图与表格清单（ICCV风格）

本文件整理你当前论文**应绘制的图/表**、放置位置、内容要求与工具建议，便于按清单逐一完成。

---

## A. 关键图（Figure）清单

### F1 方法总览图（必画）
- **目的**：一图讲清 R$^{3}$ 的核心流程与模块关系。
- **位置**：Method 开头（建议 `figure*` 跨两栏）。
- **内容要点**：
  - 输入：图像 + 问题 + 伪文本（pseudo-text）
  - 腐蚀模拟器 → 置信图（c_vis, c_text）
  - 双通道检索（文本 + 图像）
  - 三路径重建（prefix / memory / latent tokens）
  - 自适应门控融合
  - EMA teacher 一致性
- **风格**：流程图 + 颜色区分模块（注意色盲友好）。

### F2 腐蚀可视化与置信图（强烈建议）
- **目的**：证明 PMC 是“局部结构化破坏”，不是随机遮挡。
- **位置**：Method（Corruption Simulator 小节后）。
- **内容要点**：
  - 同一图像在 λ=0/0.4/0.8 下的腐蚀示例
  - 可视化置信图（c_vis）热力图
  - 文本腐蚀示例（OCR 断裂/噪声）

### F3 检索与重建示例（强烈建议）
- **目的**：展示检索确实参与重建，增强可信度。
- **位置**：Method 或 Experiments Analysis。
- **内容要点**：
  - Top-k 检索图像缩略图
  - Top-k 文本检索片段（简短）
  - 软前缀/门控权重（g_t, g_i, g_v）可视化

### F4 鲁棒性曲线（必画）
- **目的**：展示性能随腐蚀强度变化的趋势。
- **位置**：Experiments（Overall Performance）。
- **内容要点**：
  - 横轴：腐蚀强度 λ（0→0.8）
  - 纵轴：EM/F1（或 ANLS）
  - Base vs R$^{3}$ 曲线对比
  - 三个数据集各一子图（1x3 或 3x1）

### F5 幻觉率曲线/柱状图（建议）
- **目的**：突出“减少幻觉”的核心贡献。
- **位置**：Experiments 或 Analysis。
- **内容要点**：
  - Base vs R$^{3}$ 幻觉率
  - 可随 λ 增大变化（曲线）或固定高腐蚀对比（柱状）

### F6 消融贡献条形图（建议）
- **目的**：说明检索/重建/一致性各模块的作用。
- **位置**：Ablation 小节。
- **内容要点**：
  - 逐一关闭模块（text retrieval / image retrieval / prefix / memory / latent / gate / consistency）
  - 展示性能下降幅度

### F7 定性案例（必画）
- **目的**：直观展示正确与失败案例，增强可信度。
- **位置**：Analysis 或 Appendix。
- **内容要点**：
  - 图像 + 问题 + GT + Base 预测 + R$^{3}$ 预测
  - 同时展示检索证据（可简写）
  - 至少包含 1 个失败案例

## B. 关键表格（Table）清单

### T1 主结果表（必画）
- **位置**：Experiments 开头（主结果）。
- **内容要点**：
  - Base vs R$^{3}$
  - Clean + 多级腐蚀（0.2/0.4/0.6/0.8）

  - InfographicVQA：ANLS

### T2 消融表（建议）
- **位置**：Ablation 小节。
- **内容要点**：
  - 逐一去掉模块
  - 展示性能变化（Δ）

### T3 鲁棒性统计（建议）
- **位置**：主结果或附录。
- **内容要点**：
  - Relative Degradation (RD) 或平均降幅

### T4 效率/开销（可选）
- **位置**：Experiments 或附录。
- **内容要点**：
  - 额外参数量
  - 检索时间/吞吐变化

## C. 配图风格与排版要求（顶刊常见标准）
- **格式**：矢量 PDF（可缩放，文字清晰）。
- **字体**：与正文统一（Times/Helvetica）。
- **线宽**：1.0–1.5pt，避免过细。
- **配色**：色盲安全（推荐 Matplotlib `tab10`）。
- **信息密度**：尽量减少图内文字，信息放在图注。
- **表格**：使用 LaTeX `booktabs`，不使用竖线。

---

## D. 推荐工具（审稿人最易接受的图风格）

### 1) 方法流程图
- **Figma / Illustrator / Keynote**  
  输出 PDF 矢量图，布局清晰，易排版。

### 2) 曲线与柱状图
- **Matplotlib / Seaborn**  
  直接导出 PDF/EPS，数据驱动，易复现。

### 3) 表格
- **LaTeX booktabs**  
  专业感强，顶刊默认标准。
